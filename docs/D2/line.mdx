## Lines

```kroki type=d2
direction: right
  vars: {
    d2-config: {
      sketch: true
  }
  arrow: {
    target-arrowhead: {
      shape: arrow
    }
  }
  diamond: {
    target-arrowhead: {
      shape: diamond
      #style.filled: true
    }
  }
  circle: {
    target-arrowhead: {
      shape: circle
      style.filled: true
    }
  }
  dash: {
    style: {
      stroke-dash: 3
      animated: true
    }
  }
}

a -> b: arrow pointer {...${arrow}}
a -> c: diamond pointer {...${diamond}}
a -> d: circler pointer {...${circle}}
a -> e: dash {...${dash}}
```
